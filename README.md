# Free Focus ⏱

Un temporizador de concentración sin límite de tiempo, inspirado en [Pomofocus](https://pomofocus.io/) pero con una diferencia clave: **vos decidís cuándo termina tu sesión**. Al hacerlo, la app calcula automáticamente tu descanso:

```
descanso = tiempo_de_foco / 5
```

Si concentraste 50 minutos, te ganás 10 de descanso. Sin configuración, sin fricción.
